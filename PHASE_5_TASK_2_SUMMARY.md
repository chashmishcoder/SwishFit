Since MongoDB Atlas setup requires web interface interaction, you'll need to:

### Step 1: Create MongoDB Atlas Account
1. Visit: https://www.mongodb.com/cloud/atlas
2. Sign up with Google/GitHub (quickest)
3. Verify email

### Step 2: Create Production Cluster
1. Click "Build a Database"
2. Select **FREE M0 Sandbox**
3. Provider: **AWS**
4. Region: **Mumbai (ap-south-1)** for India
5. Name: `swishfit-production`
6. Click "Create"
7. Wait 1-3 minutes

### Step 3: Create Database User
1. Go to "Database Access"
2. Click "Add New Database User"
3. Username: `swishfit_admin`
4. Click "Autogenerate Secure Password"
5. **SAVE THE PASSWORD** ğŸ”
6. Permission: "Read and write to any database"
7. Click "Add User"

### Step 4: Configure Network Access
1. Go to "Network Access"
2. Click "Add IP Address"
3. Click "Allow Access from Anywhere" (adds 0.0.0.0/0)
4. Click "Confirm"

### Step 5: Get Connection String
1. Go to "Database"
2. Click "Connect" on cluster
3. Select "Connect your application"
4. Copy connection string
5. Modify:
   - Replace `<password>` with your actual password
   - Add `/swishfit` before `?`

**Example:**
```
mongodb+srv://swishfit_admin:YourPassword123@swishfit-production.xxxxx.mongodb.net/swishfit?retryWrites=true&w=majority
```
